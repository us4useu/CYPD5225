EZ-PD CCGx Host SDK

The CCGx Host SDK provides a set of firmware resources that allows users to build customized USB Type-C and Power Delivery applications using the CCGx family of Type-C port controllers from Cypress. The SDK is based on a CCGx firmware stack, which is tested and proven to be fully compliant with the USB Type-C and USB-PD specifications, and provides programming hooks and interfaces for customers to implement their own policy and system management schemes. The firmware stack provided by this version of the SDK supports the CCG2, CCG3, CCG4 and CCG5 product families and is targeted at implementing USB-PD port controllers for desktops and notebook computers.
